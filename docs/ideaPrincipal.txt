El proyecto consiste en un servidor web que meustra de manera organizada y customiable los datos reportados por un conjuto de sensores.
Tambien dispone de una API desde la cual solicitar y repotar datos nuevos

La estructura consta de 2 partes principales:

Una api que nos permite guardar y consultar datos de una base de datos relacional

Una interfaz web en la que mostrar datos en diferentes tipos de formatos y diferentes filtros de manera customizable.
Tiene una gestion basica de usuarios 

Un conjunto de sensores que son capaces de conectarse al servidor y repotar datos. Tambien han de poder recibir y relizar acciones.


Lenguajes a usar:
  Java: Servidor Web y API (SpringBoot)
  SQL : Base de datos relacional
  Micropython: Programacion de los sensores
